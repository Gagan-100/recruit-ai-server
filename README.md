# 🚀 Recruit-AI Backend

Backend API for the Recruit-AI recruitment platform built using Node.js, Express.js, and MongoDB.

---

## 🌐 Live API

Backend API: https://recruit-ai-backend-w5gm.onrender.com/

---

## 📌 Project Overview

Recruit-AI Backend handles authentication, resume uploads, job management, candidate applications, and cloud-based file storage. It provides secure REST APIs for seamless communication between frontend and backend services.

---

## ✨ Features

* 🔐 JWT Authentication
* 📄 Resume Upload API
* 🧠 Skill Extraction Logic
* 💼 Job Management APIs
* 📬 Application Handling
* ☁️ Cloudinary Integration
* 🗄️ MongoDB Database
* 📡 RESTful API Architecture

---

## 🔥 Key Functionalities

* Recruiters can create, update, and manage jobs
* Candidates can upload resumes and apply for jobs
* Resume skill extraction for better candidate filtering
* Secure authentication and protected routes
* Cloud-based resume storage using Cloudinary

---

## 🛠️ Tech Stack

* Node.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Gagan-100/recruit-ai-server.git
```

### Backend Setup

```bash
npm install
npm start
```

---

## 🔑 Environment Variables

Create `.env` file:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
PORT=5000
```
